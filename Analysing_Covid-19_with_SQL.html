Analysing Covid-19 Data with SQL

In this project, I am going to analyse Covid-19 data using SQL codes. We are going to see cases, tests and vaccination numbers up to countries. This project does not have visualisation. I am going to visualise these results on another project.

You can find all the codes I used on this project on my GitHub page: https://github.com/TurkmenOzgun/CovidStatisticsSQL/blob/main/CovidStatisticsSQL.sql


Downloaded Coronavirus data from https://ourworldindata.org/covid-deaths.


The date is between February 15, 2020, and February 22, 2022.

Here is my Excel file:



There are 163,788 rows and 67 columns. 

This data has daily info about countries' total covid cases and deaths, new cases and dates, vaccinations etc.

I erased many not needed columns and I created 2 Excel files. One is CovidDeaths.xlsx other one is CovidVaccinations.

I used SQL Server Import and Export tool and uploaded both Excel files to Microsoft SQL Server Management Studio.



Let’s start with some basic queries:

  1. Which country has the most Covid-19 cases?

Note: I used “continent is not null” because, in the dataset, there is also information about regions. For now, I am looking only at country-based data.

We can see most Covid-19 cases by country but it doesn’t make sense much because it is parallel with their population. Let’s check cases per population for more meaningful results:

  1. Total cases by population


Faroe Island, Andorra, Denmark, Gibraltar and Slovenia have the most cases according to population. It seems like small and developed countries are leading according to these numbers.


  1. Vaccinated People by Population

Let’s begin to check vaccination data:

Biggest vaccination rate by population in the United Arab Emirates.


  1. Total tests by population


When we check the test numbers per population, Austria, UAB and Denmark are at the top3. Austria made Covid-19 tests around 17 times more than their population.

  1. Total Deaths per population

Peru, Bulgaria and Bosnia and Herzegovina have the most death rates.


More Advanced SQL

In this case, I used CTE and partition by features on SQL. With this code, we can see how vaccinations went for all countries and I calculated daily increases. This result is not very readable yet but I will use these outputs in my next project “data visualization with Tableau”.

 In this case, I wanted to use a temp table. And as a result, we can see the progress of how tests going. Again, the results are not very reader friendly, but these results will be used on Tableau later.
